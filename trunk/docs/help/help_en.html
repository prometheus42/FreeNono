How to play FreeNono

FreeNono is an open-source implementation of the Nonogram game â€” a logic puzzle in which cells in a grid have to be colored or left blank according to numbers given at the side of the grid to reveal a hidden picture.

To win the the game you have to clear this hidden picture. By deducing from the given numbers you can mark (left mouse click) this fields. All fields that do not belong to the picture you can mark as empty (right mouse click). After you marked all fields of the picture correctly you win the game.

When the "h" key is pressed you get a little help solving the puzzle.
