Spielanleitung für FreeNono

FreeNono ist eine Open-Source-Umsetzung des Nonogramm-Spiels. Bei diesem Rätselspiel werden Zellen auf dem Spielfeld markiert oder leer gelassen, abhängig von den Zahlen am Rand. Als Ergebnis entsteht ein Bild.

Um zu gewinnen, muss das versteckte Bild freigespielt werden. Durch logisches Denken können die Felder bestimmt werden, die ausgefüllt werden müssen (Mausklick links). Alle nicht zum Bild gehörenden Felder können als solche markiert werden (Mausklick rechts). Wenn alle Felder korrekt ausgefüllt oder alle nicht zum Bild gehörenden Felder markiert sind, ist das Spiel gewonnen.

Bei Drücken der Taste "h" bekommst Du einen Tipp zur Lösung des Rätsels.
